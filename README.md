# Hospital-Management-System
A hospital management system using MySQL
A simple hospital management system built using MySQL to manage patient records, doctor details, appointments, and billing information

## Features

- Add, view, update, and delete patient details
- Manage doctor information (name, department, contact)
- Book and manage appointments
- Store billing and payment records
- Generate simple reports (e.g., list of patients, appointments for a day)

## Technologies Used

- **Database:** MySQL
-
